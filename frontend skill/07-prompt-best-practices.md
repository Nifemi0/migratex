# Prompt Engineering Best Practices (Frontend)

- Set role/context: "You are a senior <<framework>> engineer."
- Specify exact scope and exclusions; forbid unrelated files.
- Provide inputs: props, states, tokens, data shapes, breakpoints, accessibility bar.
- Ask for file tree plus full contents; prefer one code block per file.
- Work iteratively: component -> tests -> stories -> accessibility audit.
- Enforce quality gates: responsiveness at 360/768/1280, WCAG AA, lint/test passing.
